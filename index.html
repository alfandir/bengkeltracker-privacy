Kebijakan Privasi

Bengkel Tracker — Aplikasi Pencatat Servis Kendaraan

1. Pendahuluan

Kebijakan Privasi ini menjelaskan bagaimana Bengkel Tracker (selanjutnya disebut "Aplikasi", "Kami") mengumpulkan, menggunakan, menyimpan, dan melindungi informasi pribadi Anda saat menggunakan aplikasi kami.

Dengan menggunakan Aplikasi ini, Anda menyetujui praktik yang dijelaskan dalam Kebijakan Privasi ini. Jika Anda tidak setuju, mohon untuk tidak menggunakan Aplikasi.

2. Informasi yang Kami Kumpulkan

a. Informasi Akun

  • Nama lengkap
  • Alamat email
  • Kata sandi (disimpan dalam bentuk terenkripsi)

b. Data Kendaraan

  • Nama/merek kendaraan
  • Jenis kendaraan (motor/mobil)
  • Nomor plat kendaraan
  • Tahun kendaraan

c. Data Riwayat Servis

  • Tanggal servis
  • Jarak tempuh (kilometer)
  • Kategori servis
  • Nama bengkel
  • Deskripsi pekerjaan
  • Rincian biaya (suku cadang dan jasa)
  • Foto struk/nota servis (opsional)

d. Informasi Perangkat

  • Versi aplikasi
  • Jenis dan versi sistem operasi

3. Cara Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Menyediakan dan mengelola akun pengguna
  • Menyimpan dan menampilkan data kendaraan serta riwayat servis Anda
  • Menyinkronkan data antar perangkat melalui layanan cloud
  • Meningkatkan kualitas dan performa Aplikasi
  • Mengirim notifikasi terkait pembaruan aplikasi (jika diperlukan)

4. Penyimpanan Data

Data Anda disimpan secara aman di server Supabase yang menggunakan infrastruktur cloud dengan enkripsi standar industri. Kami menerapkan langkah-langkah keamanan yang wajar untuk melindungi data Anda dari akses tidak sah, pengubahan, pengungkapan, atau penghancuran.

Foto struk/nota yang Anda unggah disimpan di penyimpanan cloud yang aman dan hanya dapat diakses oleh akun Anda.

5. Keamanan Data

Kami berkomitmen untuk menjaga keamanan data Anda dengan menerapkan:

  • Enkripsi kata sandi menggunakan standar industri
  • Row Level Security (RLS) — setiap pengguna hanya dapat mengakses data miliknya sendiri
  • Koneksi terenkripsi (HTTPS/TLS) untuk semua transfer data
  • Autentikasi berbasis token yang aman

6. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, memperdagangkan, atau menyewakan informasi pribadi Anda kepada pihak ketiga. Data Anda hanya digunakan untuk keperluan operasional Aplikasi.

Kami dapat mengungkapkan informasi jika diwajibkan oleh hukum atau peraturan pemerintah yang berlaku.

7. Hak Pengguna

Anda memiliki hak untuk:

  • Mengakses — melihat data pribadi yang kami simpan tentang Anda
  • Memperbarui — mengubah informasi profil, kendaraan, dan data servis Anda
  • Menghapus — menghapus data kendaraan dan riwayat servis kapan saja
  • Menutup akun — menghubungi kami untuk menghapus seluruh akun dan data terkait

8. Penyimpanan Lokal

Aplikasi dapat menyimpan preferensi pengaturan (seperti tema tampilan) secara lokal di perangkat Anda. Data ini tidak dikirim ke server kami.

9. Anak-Anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak. Jika Anda mengetahui bahwa anak di bawah 13 tahun telah memberikan data pribadi kepada kami, silakan hubungi kami agar kami dapat menghapus informasi tersebut.

10. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diberitahukan melalui pembaruan aplikasi atau pemberitahuan di dalam Aplikasi. Penggunaan Aplikasi secara berkelanjutan setelah perubahan dianggap sebagai persetujuan Anda terhadap kebijakan yang diperbarui.
